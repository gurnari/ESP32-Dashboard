La police texte `epaperFont` doit contenir tous les glyphes accentués français
(é è à ç ù ê î ô …) ainsi que les ponctuations utilisées par l'UI (`«` `»` `’` `…`
`—` `→`). Le jeu de glyphes ci-dessous est passé en codes hexadécimaux pour rester
explicite et trié (le firmware fait une recherche binaire sur le CharMap, l'ordre
ascendant est obligatoire). Glyphes inclus : chiffres, ponctuation de base,
apostrophe `'`, A-Z, a-z, `~ ° ³ µ`, guillemets `« »`, accents FR minuscules et
majuscules (à â ä æ ç è é ê ë î ï ô ö ù û ü ÿ œ + capitales), ligatures œ/Œ/æ/Æ,
et symboles `— ’ … →`.
